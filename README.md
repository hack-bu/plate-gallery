# plate-gallery

Parent repository for **Bounty #1: PlateGallery** — a BU Spark! Innovation Hours bounty.

This repo is the submission target. Do not develop directly here. Fork it, build in your fork, then submit by opening a pull request back into this repo.

## How to participate

1. **Read the project brief** — [View Brief](https://docs.google.com/document/d/1gi3kzYFKG_e_6IFV3GCC1VCTyqcLZ-eEkGE8kDcfoAY/edit?usp=sharing)
2. **Read the participant guide** — [View Guide](https://docs.google.com/document/d/1gaKahLkqOo8qxo78omG2NSAmWRFBEsyKfCcLItngBhk/edit?usp=sharing)
3. **Fork this repo** and create a branch named `your-team-name/main`
4. **Build your project** in your fork
5. **Submit** by opening a PR into this repo titled `[Team Name] — PlateGallery Submission`

## Submissions
All submissions are tracked on the [Project Board](../../projects/1).
